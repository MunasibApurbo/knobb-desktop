# Code of Conduct

## Our Standard

We want Knobb to be an open, respectful project. Contributors are expected to:

- be respectful in disagreement
- give technical feedback directly and constructively
- assume good intent unless there is clear reason not to
- avoid harassment, discrimination, and personal attacks

## Unacceptable Behavior

This includes:

- abusive or insulting language
- harassment or intimidation
- discriminatory remarks
- doxxing or sharing private information without permission
- repeated bad-faith disruption of discussion or review

## Scope

This code of conduct applies in project spaces, including issues, pull requests, discussions, and other official communication channels.

## Enforcement

Project maintainers may remove comments, close threads, reject contributions, or ban participation for behavior that violates this code of conduct.

## Reporting

If you experience or witness a serious problem, contact the maintainers through the project’s primary contact channel and include enough context for review.
